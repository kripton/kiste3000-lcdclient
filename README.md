# kiste3000-lcdclient

LCDd client for Kiste3000.
Replaces lcdproc in a sense that it displays system stats.
Also provides menu items (shut down the system, ...).

## Getting Started

* Run `qmake .`
* Run `make`
* Run the resulting program ;)
